About

pxtone collab (ptcollab) is a piano-roll-based music editor where you can collaborate with friends! It uses the same instrument and project file formats as pxtone, a sample-based music editor developed by Pixel, the creator of Cave Story and Kero Blaster.

Features

Multiplayer editing is the primary feature of ptcollab. But, if you've used pxtone collage before, most features you're familiar with are available here in some form:
  • Note and parameter input
  • Instrument import (ptvoice, ogg, wav, ptnoise)
  • Song parameter settings (tempo, time signature)

Install

You can download the latest version here. If you're running a different operating system, there are build instructions in the source repo, which also has changelogs and more information.

Controls

The basic workflow is similar to pxtone collage's:

  • Start a project
  • Load a number of instruments (voices)
  • Create a number of units assigned to voices
  • In the piano roll, left-click to input notes, and right-click to delete.

The primary difference is that starting a project now either starts a server or connects to one. If you want to work alone, it's reasonable to choose a port that's not open to the internet so that only you can connect to it.

In addition to the piano roll, there are certain other song parameters that come from pxtone collage:

  • Loop points, via the measure ribbon above the piano roll
  • Automation for each unit, including pan, tuning, instrument, volume, portamento via the bar below the piano roll
  • Tempo and time signature, via the Song tab
  • Echo and overdrive effects you can apply to groups of units, via the Song tab

There are also a bunch of shortcuts, some of which are in pxtone collage, that make certain actions easier:

  • Space to play/pause, Escape to stop, Shift+click to seek
  • Ctrl+Z and Ctrl+Y/Ctrl+Shift+Z to undo and redo
  • Ctrl+click to modify just pitch values, not notes (useful for portamento)
  • Selection
    • Ctrl+shift+click to select
    • Ctrl+A to select all
    • Ctrl+shift+rclick or Ctrl+D to deselect
  • While something is selected:
    • (shift) up / down to transpose or shift parameters. Add Ctrl to shift by an octave.
    • Ctrl+C and Ctrl+V to copy/paste
  • Keyboard shortcuts for navigation:
    • W/S for next previous / next unit (PgUp / PgDn also work)
    • Q/A for previous / next parameter
    • F1..F4 for the tabs in the menu
    • 1..9 for jumping to a unit directly
  • Mouse-related shortcuts for view / mode changes
    • Middle-click drag to move around
    • Shift+wheel to scroll horizontally
    • Ctrl+(shift)+wheel to zoom vertically (horizontally)
    • Alt+wheel to change quantization
